## 📋 Sobre o Projeto
O **Blog Pessoal** funciona como uma rede social de artigos e publicações. A interface foi cuidadosamente projetada com um tema visual elegante em tons de vinho/marrom escuro (`#2c1417`, `#422125`, `#522a2e`), contando com suporte completo a operações CRUD (Criar, Ler, Atualizar e Deletar) tanto para postagens quanto para temas (categorias).

### ✨ Principais Funcionalidades
- **Autenticação Completa:** Fluxo robusto de Login e Cadastro de novos usuários com persistência via `AuthContext` e validação de tokens JWT.
- **Gerenciamento de Postagens:** Criação e edição de publicações por meio de formulários dinâmicos e modais dedicados (`ModalPostagem`).
- **Filtragem por Temas:** Organização de postagens associadas a diferentes temas/categorias customizados.
- **Perfil do Usuário:** Área de perfil personalizada com opções interativas de gerenciamento de informações e stack tecnológica.
- **Feedback Visual (Toasts):** Alertas amigáveis personalizados utilizando `React Toastify` integrados ao esquema de cores da aplicação.
- **Indicadores de Carregamento:** Componentes dinâmicos com `React Spinners` (`ClipLoader` e `SyncLoader`) para melhor experiência durante requisições assíncronas via Axios.
